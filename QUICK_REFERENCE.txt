┌──────────────────────────────────────────────────────────────┐
│ 🎮 HOW IT WORKS                                              │
└──────────────────────────────────────────────────────────────┘

THROW A BOTTLE:
  1. Click "✍️ Write a Message"
  2. Type message (10-500 chars)
  3. Add name (optional)
  4. Click "🍾 Throw Bottle"
  5. Bottle appears in ocean!

FISH FOR A BOTTLE:
  1. Click "🎣 Fish for a Bottle"
  2. Enter your name (optional)
  3. Random bottle is caught
  4. Read the message
  5. Bottle removed from ocean

┌──────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                           │
└──────────────────────────────────────────────────────────────┘

Problem: Bottles not showing
Fix: Check Firebase config, enable Firestore

Problem: Permission denied
Fix: Use Firestore "test mode"

Problem: JavaScript errors
Fix: Open browser console (F12)

Problem: Styles not loading
Fix: Check file paths, clear cache

┌──────────────────────────────────────────────────────────────┐
│ 📊 DATABASE STRUCTURE                                        │
└──────────────────────────────────────────────────────────────┘

bottles (collection)
  └── {bottleId} (document)
      ├── message: string
      ├── author: string
      ├── thrownAt: timestamp
      ├── isCaught: boolean
      ├── caughtBy: string|null
      └── caughtAt: timestamp|null
